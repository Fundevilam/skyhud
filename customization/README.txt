/////////////////////////////////////////////////
//HUD CROSSHAIRS
/////////////////////////////////////////////////

-Check "TF2Crosshairs.png" to choose your crosshair and respective character
-Open "Crosshairs.res" and follow the instructions written there


/////////////////////////////////////////////////
//HITMARKERS
/////////////////////////////////////////////////

The hitmarkers are enabled by default and you can change their color on "colors.res" at "Crosshair Pulse" using an RGB code (for example, red would be "255 000 000 255").
If you want to disable it just change the color  to "0 0 0 0".


/////////////////////////////////////////////////
//COLORS
/////////////////////////////////////////////////

With "colors.res" you can change the color of various hud elements using RGB code.
